Dark Mode Keyboard Shortcuts Test

Test Instructions

Keyboard Shortcut: Press Ctrl+Shift+D (Windows/Linux) or Cmd+Shift+D (Mac) to toggle dark mode.

Expected Behavior:

  • Dark mode should toggle when pressing the keyboard shortcut
  • FAB icon should animate (rotate 360 degrees)
  • Shortcut should NOT work when typing in input fields
  • Visual feedback should be immediate

Test Cases

Test 1: Basic Keyboard Shortcut Pending

Press Ctrl+Shift+D or Cmd+Shift+D to toggle dark mode.

Test 2: Shortcut Disabled in Input Fields Pending

Type in the input field below and press the keyboard shortcut. Dark mode should NOT toggle.

Test 3: Shortcut Disabled in Textarea Pending

Type in the textarea below and press the keyboard shortcut. Dark mode should NOT toggle.

Test 4: Shortcut Disabled in Select Pending

Focus the select dropdown and press the keyboard shortcut. Dark mode should NOT toggle.

Test 5: FAB Animation on Keyboard Toggle Pending

Press the keyboard shortcut and observe the FAB icon. It should rotate 360 degrees.

Test 6: Multiple Rapid Toggles Pending

Press the keyboard shortcut multiple times rapidly. Each toggle should complete before the next starts.

Test 7: Cross-Platform Modifier Keys Pending

Test both Ctrl (Windows/Linux) and Cmd (Mac) modifiers work correctly.
