## 🚀 Jak Uruchomić (3 kroki)

### 1. Przejdź do katalogu
```bash
cd tests/visual-testing
```

### 2. Uruchom skrypt
```bash
./run-tests.sh
```

### 3. Kliknij "Uruchom Wszystkie Testy"
Przeglądarka otworzy się automatycznie!
